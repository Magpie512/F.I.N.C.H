# F.I.N.C.H - Functionally Intelligent Networked Companion House-bot

This repository contains the custom code, configuration, and documentation for my **F.I.N.C.H** autonomous robot, which is based on the **SunFounder Pi-Crawler Kit**.

## Documentation and Setup

The foundational hardware assembly and initial setup instructions are derived from the official kit documentation.

* **Official Pi-Crawler Documentation:** [https://docs.sunfounder.com/projects/pi-crawler/en/latest/](https://docs.sunfounder.com/projects/pi-crawler/en/latest/)
